- **Fork** the Repository
- Open your **Terminal / Cmd**
 <br><br>
 ```
 git clone "https://github.com/vvHacker007/Mind-Space.git"
 ```   
 
 ```
 cd Mind-Space
 ```
 
 ```
 pip install virtualenv
 python -m venv <name of environment>
 source <name>/bin/activate
 pip install -r requirements.txt
 ```
 
 ```
  ./run
 ```
